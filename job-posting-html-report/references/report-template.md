# Report Template

Generate `index.html` with short English navigation labels and Korean section content.

## Navigation

- `Overview`
- `Role`
- `Company`
- `Fit`
- `Strategy`
- `Sources`

## Overview: 한 페이지 요약

Include:

- 회사명
- 포지션
- 종합 판단
- 지원 추천도
- 회사 성장성
- 회사 안정성
- 기술 조직 매력도
- 주요 리스크
- 핵심 어필 포인트
- 면접에서 반드시 확인할 질문 3개

This section must answer: `그래서 지원자는 지원해도 되는가? 무엇을 확인해야 하는가?`

## Role: 채용 공고 구조화와 포지션 분석

Include:

- 기술스택
- 주요 업무
- 자격 요건
- 우대 사항
- 채용 절차
- 혜택/복지
- 근무 형태
- 근무지
- 경력 요건
- 누락 정보
- 실제 기대 역할
- 요구 역량
- 기술 난이도
- 경력 수준 추정
- 성장 가능성
- JD상 위험 신호

## Company: 회사 분석

Include:

- 제품/서비스
- 사업 모델
- 수익원
- 주요 고객/시장
- 최근 뉴스
- 재무 상태 또는 투자 현황
- 성장 동력
- 회사 리스크

## Fit: 지원자 관점 판단

Include:

- 지원 적합도
- 기술스택 매칭도
- 커리어 성장 가능성
- 회사 안정성
- 회사 성장성
- 기술 조직 매력도
- 지원 시 리스크
- 지원 전 확인해야 할 점

If optional candidate information is missing, state that the fit analysis is generalized.

## Strategy: 지원 전략

Include:

- 회사 관심도를 어필할 포인트
- 지원 동기 재료
- 이력서/자소서에 반영할 키워드
- 면접에서 사용할 회사 리서치 기반 질문
- 면접에서 확인해야 할 리스크 질문
- 합류 전 판단 기준

Do not exaggerate motivation material. Connect it to actual products, business model, public values, news, or engineering direction.

## Sources: 출처

Include:

- 채용 공고 URL
- 회사 공식 자료
- 재무/공시/투자 자료
- 뉴스 자료
- 기술/채용 블로그
- 기타 참고 링크

For each source, include title, URL, owner or publisher, accessed date, and what claim it supports.
